# Intelligent Log Analyzer

An AI-ready log analysis platform built with ASP.NET Core and Blazor for Production Support Engineers.

## Features

- Upload application logs
- Detect HTTP errors
- Parse exceptions
- Group duplicate errors
- Timeline analysis
- Rich metadata extraction (in progress)
- AI-powered Root Cause Analysis (planned)

## Technology Stack

- .NET 9
- Blazor Server
- ASP.NET Core
- C#
- Git
- GitHub
